# Operations Docs

This folder contains the recovered operational state, audit, handoff, and
phase-execution documentation for the live lab environment.

## Files

- `AUDIT.md`: operational audit of the completed lab
- `HANDOFF.md`: session handoff notes and execution guidance
- `LAB-COMPLETE.md`: consolidated complete-state reference
- `LAB-CONTEXT.md`: quick operational context and connection reference
- `LAB-PHASE-PLAN.md`: recovered phase planning document
- `LAB-PHASE-RUNBOOK.md`: recovered phase execution runbook
- `LAB-STATE.md`: detailed session/build state log
- `PHASES-8-11-COMPLETION.md`: completion notes for phases 8-11
- `PHASES-8-11-READINESS.md`: readiness notes for phases 8-11

## Notes

- `LAB-PHASE-PLAN.md` and `LAB-PHASE-RUNBOOK.md` were reconstructed from
  surviving project artifacts during recovery.
- These docs describe the later operational state of the lab and complement the
  build-oriented guides in the parent `docs/` folder.
